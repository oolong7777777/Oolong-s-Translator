⭐ This is a PDF Smart Comparison Translation Tool
– Supports uploading PDF documents for automatic segmentation and translation. 
– Users can manually configure the API key of the AI translation model to be used.

⚠️ Security Notice: 
– This is a client-side only tool. Your API Key is stored in your browser's localStorage and sent directly to the translation service providers. 
– ❗ Do not use on shared devices. 
– ❗ It is recommended to rotate your API Key regularly.
– ❗ Consider using a backend proxy for team or production use.
